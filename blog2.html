What are the different HTTP requests ?

July 21,2021

What is an HTTP request ? An HTTP request basically is an action performed on a resource identifies by a given Request URL. There are various HTTP request methods and each one is assigned a speicific purpose. Request methods are case sensitive and hence are always written in upper case.


How do HTTP requests work?

HTTP requests are an intermediate transportation method between a client and a server. What actually happens is that a client sends a request(HTTP request) to the server, and the server processes that request and returns a back a response about the asked request. So in this way as we can notice, an HTTP request helps in communication between client and server.

Different types of HTTP requests.

1. GET Many of us might have used GET request while working with forms.
So what actually a GET request is?
BAsically, a GET request requests the data from a specific resource to which the GET request is sent. GET request is sent in header, therefore only a limited amount of data can be sent. GET request should not be used for sending secure data because the data is exposed in the URL bar. When we make two GET requests one after another, the second request will not be considered unless the first request is delivered. So this is how a GET request works.
