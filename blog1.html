Basics Of Javascript

As i got started with the videos, the first task was to learn basics of Javascript and get started toward making of my First CLI App. Following the video, we started with the very basic that what a variable is and how to declare it.

  1. How to declare a variable

    2. var = (value);

  2. How to take input from user

    3. var username = readlineSync.question("What is your name ?")

  3. How to output something

    4. console.log("Hello")

  4. If else statement

    5. if (condition) { // block of code to be executed if the condition is true } else { // block of code to be executed if the condition is false }

  5. For loop

    6. for (statement 1; statement 2; statement 3) { // code block to be executed }

  6. Functions

    7. function name(parameter1, parameter2, parameter3) { // code to be executed }
